Recommendation for Option B Layout
💡 Recommended: Variation 1 (Chips)
Best fit for Option B's side-by-side layout:
  • Compact dropdown keeps the 220px "Apply to" column clean
  • Shows summary when closed ("All" or "Specific + count")
  • Opens to reveal radio + chips for easy selection
  • Chips are more visual and easier to toggle than checkboxes
  • Works well in the constrained horizontal space
Alternative: Variation 2
Checkboxes if accessibility is a priority or you have 6+ campaigns.
Alternative: Variation 3
Quick actions if users frequently select all/none.
