USI Multi-Select Campaign Rules - UI Explorations

Supporting "All Campaigns" vs "Specific Campaigns" with multi-select

Option 1: Radio Buttons + Chip Multi-Select

Pattern: Radio buttons for "All" vs "Specific", with chip selector appearing inline when "Specific" is selected.
Best for: Visual clarity, easy to understand, good for 2-6 options.
Interaction: Click chips to toggle selection. Selected chips change color/border.

Option 2: Radio Buttons + Checkbox List

Pattern: Radio buttons for "All" vs "Specific", with checkbox list appearing when "Specific" is selected.
Best for: Precision, accessibility, when you have many options (6+).
Interaction: Check/uncheck boxes. More explicit than chips, better for screen readers.

Option 3: Dropdown with Popover Multi-Select

Pattern: Single dropdown showing current selection summary. Click to open popover with radio + chips/checkboxes.
Best for: Space efficiency, when you have limited vertical space.
Interaction: Dropdown shows "All campaigns" or "3 selected". Opens popover with full options.

Option 4: Segmented Control + Chip Selector

Pattern: iOS-style segmented control for "All" vs "Specific", with chip selector appearing below.
Best for: Modern, compact, when you want a cleaner visual hierarchy.
Interaction: Toggle between segments, chips appear for "Specific" mode.

Option 5: Inline Radio + Quick Action Buttons

Pattern: Radio buttons with "Select all" / "Clear all" quick action buttons next to "Specific campaigns" option.
Best for: Power users, when bulk selection is common.
Interaction: Quick actions for common patterns (all, none), plus individual chip selection.

Recommendation Summary
✅ Best Overall
Option 1: Radio + Chips
Most intuitive, visual, easy to scan. Chips are self-explanatory and work well for 2-6 options. Good balance of clarity and compactness.
✅ Best for Accessibility
Option 2: Radio + Checkboxes
Explicit, accessible, works with screen readers. Better for many options or when precision matters.
✅ Best for Space
Option 3: Dropdown Popover
Most compact. Shows summary, expands on demand. Good when vertical space is tight.
✅ Best for Modern UI
Option 4: Segmented Control
Clean, iOS-style. Good visual hierarchy. Works well in mobile-first designs.
💡 Recommended Direction
Option 1 (Radio + Chips) is the strongest choice for your use case. The chips provide:
  • Clear visual feedback of selection state
  • Easy toggle interaction (click to select/deselect)
  • Good for 2-6 campaigns (scales well)
  • Doesn't require scrolling or expansion
  • Matches Prism design patterns
Alternative: If you need more space efficiency later, Option 3 (Dropdown Popover) is a good fallback.
